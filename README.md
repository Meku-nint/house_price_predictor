# House Price Prediction

## Description
A web application to predict house prices using Django, React, and a machine learning model (scikit-learn).

## Features
- Django backend with REST API
- React frontend for user input and displaying predictions
- Machine Learning model trained with scikit-learn
- Save and load ML model with joblib
